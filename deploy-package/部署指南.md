## 🚀 部署步骤

### 第一步：部署云函数

1. 登录 [微信云开发控制台](https://cloud.weixin.qq.com/)
2. 进入你的云开发环境
3. 创建云函数 `analyzeStock`
4. 上传 `cloud-functions/analyzeStock/` 目录下的文件
5. **配置环境变量**：
   - 名称：`MINIMAX_API_KEY`
   - 值：你的 MiniMax API Key

### 第二步：部署 H5 页面

1. 在云开发控制台开启「静态网站托管」
2. 修改 `H5页面/index.html` 中的配置（第 1224 行）：
   ```javascript
   const USE_WECHAT_CLOUD = true;   // 启用云函数调用
   const USE_STREAMING = false;     // 云函数不支持流式
   ```
3. 上传 `H5页面/` 目录下的所有文件到静态托管

### 第三步：配置服务号菜单

1. 在微信公众平台配置菜单
2. 菜单类型：跳转网页
3. 填写静态托管的 H5 页面 URL

## ⚠️ 重要配置

### 云函数配置 (cloud-functions/analyzeStock/index.js)

第 18 行已配置为高速版模型：
```javascript
const MODEL = 'MiniMax-M2.7-highspeed';
```

### H5 页面配置 (H5页面/index.html)

第 1222-1224 行：
```javascript
const USE_STREAMING = true;      // 本地开发用流式
const USE_WECHAT_CLOUD = false;  // 本地开发不用云函数
```

**上线前改为：**
```javascript
const USE_STREAMING = false;      // 云函数不支持流式
const USE_WECHAT_CLOUD = true;   // 使用云函数
```

## ❓ 常见问题

**Q: 云函数调用失败**
- 检查 `MINIMAX_API_KEY` 环境变量是否配置
- 检查云函数是否成功部署
- 在云开发控制台查看日志

**Q: H5 页面显示空白**
- 确认 `USE_WECHAT_CLOUD` 已设为 `true`
- 确认云函数名称为 `analyzeStock`
- 在微信开发者工具中开启「不校验合法域名」

**Q: 反应慢**
- 云函数模式下不支持流式返回，用户需等待完整结果
- 可以考虑使用方案B（进度提示文案）
